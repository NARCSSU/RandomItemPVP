# 更新日志 / Changelog

## [2.1.0] - 2025-11-01

### 🎉 新增功能 / New Features
- **游戏出生点持久化** - 出生点现在保存在配置文件中，重启服务器不会丢失
  - 使用 `/ripvp setspawn` 设置出生点会自动保存到 `config.yml`
  - 启动时自动从配置文件加载出生点
  - `/ripvp reload` 命令支持重新加载出生点

### 🐛 Bug修复 / Bug Fixes
- **修复严重卡顿问题** - 优化了竞技场清理逻辑，避免扫描超过1000万个方块
  - 使用流体追踪列表替代全局扫描
  - 清理耗时从5-10秒降低到<0.1秒
  - 性能提升超过100倍
- **修复床爆炸死亡后无法放置方块** - 添加重生事件监听，确保死亡玩家保持旁观者模式
- **修复部分玩家无法放置方块** - 添加存活玩家检查和在线状态验证
- **修复游戏倒计时显示不全** - 所有玩家现在可以在整个倒计时过程中看到标题提示
  - 10秒以上：副标题提示
  - 10-4秒：主标题+副标题+音效
  - 最后3秒：超大金色标题+强音效

### ⚡ 优化改进 / Improvements
- **空投打开提示增强**
  - 添加全服大标题显示（2秒）
  - 双重音效提示（全服可听到）
  - 增强粒子特效（图腾+烟花）
  - 打开者获得特殊成就音效
- **方块放置权限优化**
  - 只允许存活参与者放置方块
  - 非存活玩家会收到明确提示
  - 添加水桶和岩浆桶使用限制
- **出生点设置提示改进** - 显示更详细的坐标和保存状态信息

### 📝 配置文件更新 / Config Changes
- 新增 `arena.spawn` 配置项用于保存游戏出生点
  - `world` - 世界名
  - `x`, `y`, `z` - 坐标
  - `yaw`, `pitch` - 视角
