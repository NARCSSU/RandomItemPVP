# RandomItemPVP v2.1.0

## 🎉 重要更新

### 🎯 游戏出生点持久化
出生点现在保存在配置文件中，重启服务器不会丢失！使用 `/ripvp setspawn` 设置后自动保存。

### ⚡ 性能优化 - 修复严重卡顿
修复了游戏结束时整个服务器卡住5-10秒的问题，性能提升超过100倍！

### 🐛 重要Bug修复
- ✅ 修复床爆炸死亡后无法放置方块
- ✅ 修复游戏倒计时普通玩家看不到
- ✅ 修复部分玩家无法放置方块
- ✅ 增强空投打开提示（标题+音效+特效）

## 📦 下载与安装

### 要求
- **服务器**: Paper/Folia 1.21+
- **Java**: Java 21+

### 安装
1. 下载 `RandomItemPVP-2.1.0.jar`
2. 放入 `plugins` 文件夹
3. 重启服务器
4. 使用 `/ripvp setspawn` 设置游戏出生点

### 从 v2.0.1 升级
直接替换 jar 文件，重启后使用 `/ripvp setspawn` 重新设置出生点即可。
